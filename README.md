# CloudDrive-Personal-Notes

alfred 管理 CloudDrive 專案的個人筆記庫（日誌、harness 筆記、目標追蹤、報告）。

- **線上閱讀（GitHub Pages）**：<https://alfredshen01.github.io/CloudDrive-Personal-Notes/>
- 內容入口：[docs/index.md](docs/index.md)
- push 到 `main` 後由 GitHub Actions 自動重建網站（`mkdocs.yml`＋Material 主題）；HTML 為 build 產物，不進版本庫。
- 寫作風格由 note-style 協議把關：`vale <檔案>` 跑禁詞掃描（規則在 `.vale.ini`＋`.style/AlfredNotes/Colloquial.yml`）。

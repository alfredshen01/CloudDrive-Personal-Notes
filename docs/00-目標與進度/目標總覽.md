## 總目標

建成 CloudDrive 的對話式 AI 助理（harness），通過 V0.2 驗收（V0.2＝本專案第二版需求與驗收計畫書）。

我負責的範圍：FR-18～FR-21（對話操作、計畫確認、skill 生成、skill 管理）、NFR-07（本地模型優先與隱私）、第八章 evaluation harness。

階段劃分：研究 → 實作 → mock 驗證 → 真模型驗證 → 驗收。**目前所在階段：驗收準備**——主線功能與 mock 層已完成，缺口集中在真模型與端到端證據。

## 本專案的功能邊界

- 不換模型、不以雲端 API 為主力——本地優先、隱私保護是本專案的根本定位（DEC-018）。
- 不做 agentic loop 與 DAG 並行——已正式議決（DEC-029/030）,無新數據不重開。

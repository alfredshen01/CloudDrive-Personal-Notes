## 總目標

建成 CloudDrive 的對話式 AI 助理（harness），通過 V0.2 驗收。

我負責的範圍：FR-18～FR-21（對話操作、計畫確認、skill 生成、skill 管理）、NFR-07（本地模型優先與隱私）、第八章 eval harness。

階段劃分：研究 → 實作 → mock 驗證 → 真模型驗證 → 驗收。**目前所在階段：驗收準備**——主線功能與 mock 層已完成，缺口集中在真模型與端到端證據。

| 搜尋 backfill 未背景自動化 | 低 | — |
| 時光機還原硬配額檢查待補 | 低 | — |

遠期方向：記憶 v2（摘要壓縮 → 語意檢索復用 pgvector → 跨 session）。

## 明確不做（範圍邊界,避免發散）

- 不換模型、不以雲端 API 為主力——本地隱私主軸是專案識別（DEC-018）。
- 不做 agentic loop 與 DAG 並行——已正式議決（DEC-029/030）,無新數據不重開。
- 驗收報告前不開新的大功能面——剩餘時間投「可靠性證據＋記憶體驗」報酬率最高。

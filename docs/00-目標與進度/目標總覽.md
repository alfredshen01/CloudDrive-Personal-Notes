## 總目標

建成 CloudDrive 的對話式 AI 助理（harness），通過 V0.2 驗收（V0.2＝本專案第二版需求與驗收計畫書）。

我負責的範圍：FR-18～FR-21（對話操作、計畫確認、skill 生成、skill 管理）、NFR-07（本地模型優先與隱私）、第八章 evaluation harness。

階段劃分：研究 → 實作 → mock 驗證 → 真模型驗證 → 驗收。**目前所在階段：驗收**——主線功能、mock 層、全案例真模型評測皆已完成；缺口集中在**端到端整合測試**、**外部升級（NFR-07）**，以及困難級別（M5）的模型能力上限。

## 本專案的功能邊界

- **助理的職責範圍**：透過既定工具在這個雲端硬碟內操作——自然語言 → 規劃工具步驟 → 寫入前確認 → 執行檔案操作／現場生成並執行技能（對應 FR-18～21）。定位是「操作這個硬碟的助手」，不是通用聊天機器人。
- **本地優先**：模型預設用本地模型、資料不外流；外部模型由使用者逐訊息自選（DEC-018／NFR-07）。
